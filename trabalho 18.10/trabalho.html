Definição de parâmetros para o Imput:

A anotação input torna possível modificar valores específicos que serão usados na
lógica do algoritmo sem a necessidade de alterar o código fonte.

1-Autocomplete

Especifica se um elemento input deve ter o preenchimento automático ativado ou não.

2, 3 - Max e Min

O atributo HTML min é usado para especificar o valor minimo para um campo de entrada.
pode ser usado juntamente com o atributo máximo para criar uma gama de valores.
Ele pode ser usado com os muitos campos de entrada, como número, faixa, data, data, data-hora local, mês, hora e semana.
